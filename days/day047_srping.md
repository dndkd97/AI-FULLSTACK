🛠️ 팀원 작업 흐름
1. 저장소 클론하기
git clone <팀장이 만든 저장소 주소>
cd <저장소 폴더>

2. 브랜치 생성하기
git checkout -b feature-브랜치이름

3. 작업 후 커밋하기
git add .
git commit -m "작업 내용 설명"

4. 원격 저장소에 브랜치 푸시하기
git push origin feature-브랜치이름
